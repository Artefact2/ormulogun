Ormulogun
=========

A very simple serverless, privacy aware chess tactics
trainer. Released under the Apache License 2.0.

Test it live at: <https://artefact2.gitlab.io/ormulogun/>

Dependencies
============

* PHP (CLI)
* Any UCI engine (eg stockfish)
* CMake
* LLVM, clang
* Emscripten
